# Ejercicio 4 — Notas de estudiantes

## Desarrollo del Problema

### Análisis del problema

En un curso hay n estudiantes.

Se necesita:

1. Leer cantidad de estudiantes.
2. Leer nota de cada estudiante.
3. Guardar las notas en un arreglo.
4. Contar notas entre 3.0 y 4.0.
5. Contar notas mayores a 4.0.
6. Calcular la suma total de notas.

### Resultados esperados

* cantidadNotas3Y4
* cantidadNotasMayores4
* totalNotas
