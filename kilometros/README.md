# Ejercicio 9 — Kilómetros recorridos

## Desarrollo del Problema

### Análisis del problema

Una empresa registra kilómetros recorridos por n vehículos.

Se necesita:

1. Leer cantidad de vehículos.
2. Leer kilómetros de cada vehículo.
3. Guardar en arreglo.
4. Contar entre 10 y 300 km.
5. Contar mayores a 300 km.
6. Calcular total recorrido.

### Resultados esperados

* cantidadVehiculos10Y300
* cantidadVehiculosMayores300
* totalKilometros
