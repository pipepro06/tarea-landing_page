# Ejercicio — Sueldos de empleados

## Desarrollo del Problema

### Análisis del problema

Una empresa tiene n empleados. Cada empleado gana un sueldo entre $100 y $500.

Se necesita crear un programa que:

1. Lea la cantidad de empleados.
2. Lea el sueldo de cada empleado.
3. Almacene los sueldos en un arreglo.
4. Cuente cuántos empleados ganan entre $100 y $300.
5. Cuente cuántos empleados ganan más de $300.
6. Calcule el total que la empresa gasta en sueldos.

### Resultados esperados

* cantidadEmpleados100Y300
* cantidadEmpleadosMayores300
* totalSueldos

### Datos disponibles

#### Datos de entrada

* cantEmpleados
* sueldo

#### Variables necesarias

* sueldos[]
* cantidadEmpleados100Y300
* cantidadEmpleadosMayores300
* totalSueldos

### Proceso lógico

1. Solicitar cantidad de empleados.
2. Crear arreglo.
3. Inicializar contadores.
4. Leer cada sueldo.
5. Guardarlo en el arreglo.
6. Clasificar.
7. Acumular total.
8. Mostrar resultados.
