# Ejercicio 8 — Gastos de clientes en un restaurante

## Desarrollo del Problema

### Análisis del problema

Un restaurante atiende n clientes.

Se necesita:

1. Leer cantidad de clientes.
2. Leer gasto de cada cliente.
3. Guardar en arreglo.
4. Contar gastos entre 20000 y 100000.
5. Contar gastos mayores a 100000.
6. Calcular total recibido.

### Resultados esperados

* cantidadClientes20kY100k
* cantidadClientesMayores100k
* totalRecaudado
