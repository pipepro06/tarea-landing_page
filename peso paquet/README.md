# Ejercicio 6 — Peso de paquetes

## Desarrollo del Problema

### Análisis del problema

Una empresa de envíos procesa n paquetes. Cada paquete pesa entre 1 kg y 50 kg.

Se necesita crear un programa que:

1. Lea la cantidad de paquetes.
2. Lea el peso de cada paquete.
3. Almacene los pesos en un arreglo.
4. Cuente cuántos paquetes pesan entre 1 kg y 20 kg.
5. Cuente cuántos paquetes pesan más de 20 kg.
6. Calcule el peso total transportado.

### Resultados esperados

* cantidadPaquetes1Y20
* cantidadPaquetesMayores20
* totalPeso

### Datos disponibles

#### Datos de entrada

* cantidadPaquetes
* peso

#### Variables necesarias

* pesos[]
* cantidadPaquetes1Y20
* cantidadPaquetesMayores20
* totalPeso

### Proceso lógico

1. Solicitar cantidad de paquetes.
2. Crear arreglo.
3. Inicializar contadores.
4. Leer cada peso.
5. Guardar en arreglo.
6. Clasificar.
7. Acumular.
8. Mostrar resultados.
