# Ejercicio 5 — Consumo de agua

## Desarrollo del Problema

### Análisis del problema

Se registra el consumo mensual de agua de n apartamentos.

Se necesita:

1. Leer cantidad de apartamentos.
2. Leer consumo de cada apartamento.
3. Guardar en arreglo.
4. Contar consumos entre 1 y 50.
5. Contar consumos mayores a 50.
6. Calcular consumo total.
