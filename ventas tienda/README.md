# Ejercicio 3 — Ventas de una tienda

## Desarrollo del Problema

### Análisis del problema

Una tienda registra n ventas durante el día. Cada venta está entre $50 y $1000.

Se necesita crear un programa que:

1. Lea la cantidad de ventas.
2. Lea el valor de cada venta.
3. Almacene las ventas en un arreglo.
4. Cuente cuántas ventas están entre $50 y $500.
5. Cuente cuántas ventas son mayores a $500.
6. Calcule el total vendido.

### Resultados esperados

* cantidadVentas50Y500
* cantidadVentasMayores500
* totalVentas
