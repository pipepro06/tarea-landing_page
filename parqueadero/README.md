# Ejercicio 10 — Recaudo de parqueaderos

## Desarrollo del Problema

### Análisis del problema

Un parqueadero registra el pago de n vehículos.

Se necesita:

1. Leer cantidad de vehículos.
2. Leer pago de cada vehículo.
3. Guardar pagos en arreglo.
4. Contar pagos entre 2000 y 20000.
5. Contar pagos mayores a 20000.
6. Calcular total recaudado.

### Resultados esperados

* cantidadVehiculos2kY20k
* cantidadVehiculosMayores20k
* totalRecaudado
