# Ejercicio 7 — Producción de una fábrica

## Desarrollo del Problema

### Análisis del problema

Una fábrica registra la producción de n operarios.

Se necesita:

1. Leer cantidad de operarios.
2. Leer producción de cada operario.
3. Guardar en arreglo.
4. Contar producciones entre 10 y 100.
5. Contar producciones mayores a 100.
6. Calcular producción total.

### Resultados esperados

* cantidadOperarios10Y100
* cantidadOperariosMayores100
* totalProduccion
